# 反幻觉机制与质量检查清单

## 常见幻觉模式规避

| 模式 | 错误做法 | 正确做法 |
|-----|---------|---------|
| 期刊猜测 | "发表于Nature"（原文未提）| `[未明确给出]` |
| 贡献膨胀 | 增加未声明的贡献 | 仅列出论文声明的贡献 |
| 结果外推 | "达到所有基准的SOTA" | 引用确切基准和指标 |
| 作者专长 | "计算机视觉专家" | 仅陈述论文声明的内容 |
| 未来工作编造 | 发明未来方向 | 仅引用论文自己的Future Work |
| 方法细节填充 | 猜测架构细节 | 标注为 `[未明确给出]` |
| 归因错误 | 将观点归于错误作者 | 确认观点的真正来源 |
| **文献ID混乱** | 使用paper_01等临时编号 | **必须使用DOI或PMID作为唯一标识** |
| 分析表面化 | `[模型归纳]` 仅停留在表面总结 | 纵向深入，层层递进分析因果链和根因 |

## 反幻觉检查 ⭐

- [ ] 所有观点都有 `[原文声明]` 或 `[模型归纳]` 标签
- [ ] 所有原文引用都使用原文英文
- [ ] 所有不确定信息都标注为 `[未明确给出]` 或 `[不确定]`
- [ ] 没有编造任何论文未声明的内容
- [ ] 没有外推论文未支持的结论
- [ ] 没有猜测缺失的信息

## 文献可追溯性 ⭐

- [ ] 每篇文献都有完整引用格式
- [ ] 每个观点都有来源文献ID
- [ ] 每个观点都有准确DOI/PMID标识（仅需ID，不需段落位置）
- [ ] 参考文献索引完整
- [ ] **所有文献引用使用DOI或PMID（禁止paper_01临时编号）**
- [ ] 信息缺失使用 `[未明确给出]`

## 分析深度检查 ⭐

- [ ] 每条 `[原文声明]` 都有紧随的 `[模型归纳]` 分析
- [ ] 不存在连续3条以上纯 `[原文声明]` 无 `[模型归纳]` 间隔
- [ ] `[模型归纳]` 均说明了推理依据
- [ ] 关键发现至少达到分析Level 2（跨文献关联）
- [ ] 鼓励核心观点达到分析Level 3（深层推理）
- [ ] **主题分析纵向深入**：不止于"X方法有效"，需分析因果链、根因、趋势推演

## 每主题深度覆盖检查

- [ ] 每个识别的主题有独立的深度分析章节
- [ ] 每个主题包含完整的8个组成部分（定义、时间线、方法表、争议、缺口、观点图、文献、未来方向）
- [ ] 每个主题至少有3个方法论演进时间节点
- [ ] 每个主题识别了至少1个争议点和至少2个研究缺口
- [ ] 每个主题的观点依赖图使用Mermaid graph语法

## 可视化质量检查

- [ ] 所有关系图使用Mermaid语法（无ASCII/纯文本替代）
- [ ] 每个Mermaid图节点包含原文引用和来源标注
- [ ] 每个Mermaid图包含颜色编码和关系类型标注
- [ ] 每个图表都有图例说明

## 内容完整性

- [ ] 时间线包含所有关键事件
- [ ] 论战分析明确指出对立观点
- [ ] 研究缺口分类清晰
- [ ] Mermaid语法正确
- [ ] 表格格式正确
- [ ] 所有主题都已识别并分类
- [ ] 主题关系图完整
- [ ] 观点依赖图显示所有关系
- [ ] **不包含"文献逐篇分析"章节（该章节已移除，易产生幻觉）**
- [ ] 所有文献标识使用DOI/PMID（非paper_XX临时编号）
